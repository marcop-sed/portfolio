# 🛠️ Cyber Engineering & Offensive Security Portfolio

Questo repository ospita l'infrastruttura e i contenuti del mio portfolio professionale. Il progetto è realizzato seguendo la filosofia **Docs-as-Code**, dove la documentazione tecnica viene trattata con lo stesso rigore del software: versionamento, automazione e strutturazione semantica.

> **Live View:** [marcop-sed.github.io/portfolio/](https://marcop-sed.github.io/portfolio/)

---

## 🏗️ Tech Stack & Architecture

Il portfolio è costruito per essere leggero, performante e facilmente mantenibile:

- **Engine:** [MkDocs](https://www.mkdocs.org/) con tema [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/).
- **Multi-language:** Gestione integrata IT/EN tramite plugin `mkdocs-static-i18n`.
- **Infrastructure:** Hosting su GitHub Pages.
- **Reporting Standard:** I report seguono le metodologie **PTES** e **NIST 800-115**, con scoring **CVSS v3.1**.
